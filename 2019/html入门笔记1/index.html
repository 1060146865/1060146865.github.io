HTML入门笔记1

Written by 赵凯 with ♥ on August 24, 2019 in

1.HTML 是谁发明的

HTML 的全名是“超文本标记语言”（HyperText Markup Language），上个世纪 90 年代由欧洲核子研究中心的物理学家蒂姆·伯纳斯-李（Tim Berners-Lee）发明。它的最大特点就是支持超链接，可以跳转到其他网页，从而构成了整个互联网。1999 年，HTML 4.01 版发布，成为广泛接受的 HTML 标准。2014 年，HTML 5 发布，这是目前正在使用的版本。

2.HTML 起手应该写什么

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Document</title>
  </head>
  <body></body>
</html>

<!DOCTYPE html>表示文档类型，告诉浏览器如何解析网页。

<html lang="en">表示网页内容的语言，可以改成zh-CN。

<head>

<meta charset="UTF-8" />文件的字符编码。

<meta name="viewport" content="width=device-width, initial-scale=1.0" />禁用缩放，兼容手机。

<meta http-equiv="X-UA-Compatible" content="ie=edge" />告诉 IE 使用最新的内核。

<title>Document</title>设置网页标题。

</head>

<body></body>浏览器显示的页面内容。

</html>标签是网页的顶层容器，也称为根元素。

3.常用的表章节的标签

<h1>~<h6>用来表示文章的标题。按照标题的等级，一共分成六级。

<header>表示整个网页的头部。

<footer>网页的的尾部。

<main>标签表示页面的主体内容。

<article>标签表示页面里面一段完整的内容。

<aside>旁支内容。

<section>表示一个章节。

<div>是一个通用标签，表示一个区块。

<p>表文章的一个段落.

4.全局属性

全局属性（global attributes）是所有元素都可以使用的属性。也就是说，你可以把下面的属性，加在任意一个网页元素上面，不过有些属性对某些元素可能不产生意义。

id 属性是元素在网页内的唯一标识符。

class 属性用来对网页元素进行分类。如果不同元素的class属性值相同，就表示它们是一类的。

title属性用来为元素添加附加说明。

contenteditable属性允许用户修改内容。

hidden是一个布尔属性，表示当前的网页元素不再跟页面相关，不会在网页中看到它。

style属性用来指定当前元素的样式。

tabindex可以使用键盘 Tab 键,选定焦点元素进行下一步操作。

5.常用的内容标签

<br>让网页产生一个换行效果。该标签是单独使用的，没有闭合标签。

<hr>分隔水平线。该标签是单独使用的，没有闭合标签。

<pre>是一个块级元素，表示保留原来的格式。只保留空格和换行，不会保留 HTML 标签。

<strong>加粗显示内容。

<em>斜体

<blockquote>表引用。

<q>表引用，但不会产生换行。

<code>表示标签内容是计算机代码。

<ol>标签是一个有序列表容器。

<ul>标签是一个无序列表容器。

<li>表示列表项，用在<ol>或<ul>容器之中。

<dl>，<dt>，<dd>。 <dl>表示一组术语的列表。术语名由<dt>标签定义，术语解释由<dd>标签定义。<dl>常用来定义词汇表。

<a>标签内可以放置链接，文字，段落、图像、多媒体等等。
