如何用hugo搭建个人博客

Written by 赵凯 with ♥ on August 29, 2019 in

1.下载Hugo

下载网站:https://github.com/gohugoio/hugo/releases/tag/v0.48

Windows下载hugo_xxx_Windows-64bit.zip

新建一个文件夹解压缩得到一个hugo.exe

2.Hugo环境变量配置

将hugo.exe的地址加到PATH里

终端下输入<hugo version>检验是否配好环境变量

3.创建一个新站点

hugo new site quickstart这里的quickstart可以改成其他的名字

4.添加主题

在站点的目录下打开终端执行以下代码:

cd quickstart

git init

git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 这里是默认主题可以下载其他的主题

echo 'theme = "ananke"' >> config.toml如果下载了其他的主题这里的ananke改成其他的主题的名字

5.添加内容

在站点的目录下打开终端执行以下代码:

hugo new posts/XXXX.md

默认文件里会有这些:

---
title: "XXXX"
date: 2019-03-26T08:47:11+01:00
draft: true
---

hugo server -D可以预览草稿

hugo server可以预览非草稿

浏览器打开http://localhost:1313，可以查看到运行效果

在站点运行hugo

这个hugo命令执行完后，会生成一个public文件夹，创建自己的GithubPages，新建仓库repository，命名为Username.github.io，在将public文件夹内的所有文件推上仓库里就可以了
