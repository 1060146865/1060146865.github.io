JS 函数的执行时机

Written by 赵凯 with ♥ on July 21, 2020 in

1.为什么如下代码会打印 6 个 6

let i = 0
for(i = 0; i<6; i++){
  setTimeout(()=>{
    console.log(i)
  },0)
}

因为setTimeout的意思是稍后执行，也就是在等for循环结束之后再执行console.log(i)，当for结束循环的时候i等于6，且只有1个i，所以打印出6个6

2.如何让以上代码打印出0–6

for(let i=0;i<6;i++){
setTimeout(()=>{
console.log(i)
},0)
}

因为JS在for和let一起用的时候会在每次循环的时候多创建一个i，每次循环都会重新创建一个i，所以会打印出0，1，2，3，4，5


let i
for(i = 0; i<6; i++){
    const x = i
    setTimeout(()=>{
      console.log(x)
    })
}



let i 
for(i = 0; i<6; i++){
  !function(j){
      setTimeout(()=>{
        console.log(j)
      },0)
  }(i)
}



for(var i=0;i<6;i++){
            (function(i){
                setTimeout(()=>{console.log(i)},0)
            })(i)
        }


let i = 0
for(i = 0; i<6; i++){
setTimeout((i)=>{
console.log(i)
},0,i)
}
